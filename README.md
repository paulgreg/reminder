# Reminder

Simple PWA in react to create reminders, inspired by slackbot command `/remind`

https://paulgreg.me/reminder

## Deploy

To deploy to github on branch gh_pages

`npm i && PUBLIC_URL="https://paulgreg.me/reminder" ./to_ghpages`
